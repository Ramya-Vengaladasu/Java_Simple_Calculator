# 🧮 Simple Calculator (Java)

## 📌 Project Overview

The Simple Calculator is a console-based Java application that performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.

The application provides a menu-driven interface, handles invalid user inputs using exception handling, and allows users to perform multiple calculations without restarting the program.

This project was built to strengthen Java fundamentals, object-oriented programming concepts, methods, exception handling, and user interaction through the console.

---

## 🚀 Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by Zero Validation
* Exception Handling for Invalid Inputs
* Menu-Driven Interface
* Multiple Calculations in a Single Run
* User-Friendly Console Interaction
